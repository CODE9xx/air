Безопасность

Данные под защитой уровня enterprise

Отдел продаж работает с чувствительной информацией — клиенты, цены, скидки, переговоры. CODE9 построен так, чтобы вы могли доверить эти данные без оговорок.

Главный принцип

Ваши данные не обучают AI

Изоляция данных — основа архитектуры CODE9. Ваша информация используется только для работы вашей команды. Ничего не утекает в чужие модели и не становится частью чужих ответов.

  • AI-модели не дообучаются на ваших запросах и данных
  • Каждый запрос обрабатывается в изолированной среде
  • Промпты и ответы не попадают в общие модели
  • По запросу — полный on-premise режим без интернета
Серверы в РФ
Yandex Cloud и Selectel — крупнейшие российские облака с собственной инфраструктурой. Данные не покидают страну.
Соответствие 152-ФЗ
Полная работа по закону о персональных данных. ФИО, телефоны, реквизиты клиентов — обработка строго по регламенту РФ.
End-to-end шифрование
AES-256 в хранении, TLS 1.3 при передаче. Ключи — в управляемом HSM. Данные зашифрованы всегда.
2FA + ролевой доступ
Двухфакторная авторизация, гибкая модель ролей и прав. Админ, РОП, менеджер — каждый видит только своё.
99,98% доступность
Отказоустойчивая архитектура: кластеры серверов, балансировка нагрузки, репликация БД. SLA для Enterprise.
Бэкапы + откат
Резервное копирование каждый час. Восстановление на любую точку за 30 дней. RTO — 15 минут.
Журнал AI-действий. Каждое решение AI логируется с контекстом — любое действие можно откатить и разобрать.
Частный контур. По запросу — выделенная инфраструктура или on-premise-развёртывание для критичных задач.
