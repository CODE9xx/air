# Code9 Analytics — Vision

## Одним предложением

SaaS, который подключается к amoCRM / Kommo / Bitrix24 по OAuth, делает **объективный CRM-аудит**, даёт владельцу **выгрузку** своих данных и **дашборды**, а для отделов продаж — **AI-анализ разговоров** с рекомендациями.

## Для кого

- **Владелец/РОП** малого и среднего бизнеса в СНГ (RU) и международного (EN), у которого amoCRM/Kommo/Bitrix24 уже работает, но данные «как будто есть, а толку ноль».
- **Внешний аудитор/интегратор**, которому нужно быстро получить срез гигиены чужого CRM.

## Ключевые сценарии MVP

1. **Подключил CRM за 3 клика** → получил отчёт «что не так» (просроченные задачи, дубли контактов, брошенные сделки, пропуски обязательных полей).
2. **Скачал всю свою базу** ZIP-архивом (CSV + JSON) — для переезда, бэкапа, внешнего анализа.
3. **Посмотрел дашборд** — воронка, конверсии по этапам, активность менеджеров, брошенные сделки.
4. **Прогнал разговоры через AI** — получил оценку каждого звонка/чата, типовые ошибки, точки роста.
5. **Дал согласие на анонимное исследование** → помогает улучшать бенчмарки индустрии, видит агрегированные отраслевые инсайты.

## Что НЕ делаем в MVP

- Не правим данные в CRM (read-only).
- Не импортируем файлы/картинки из CRM.
- Не храним аудио-записи разговоров постоянно.
- Не делаем реальных звонков/писем/CRM-изменений.
- Нет LiveChat/поддержки через виджет — только email-тикеты.

## Ценностные дифференциаторы

- **Честный аудит, а не маркетинговый отчёт.** Всё метрики объясняются, формула видна.
- **Мультитенант через изоляцию на уровне Postgres schema** — данные клиентов не смешиваются даже случайно.
- **Приватность по умолчанию.** OAuth-токены зашифрованы at rest (Fernet), destructive-действия — через email-код, аудио не хранится, AI работает на анонимизированных паттернах.
- **Mock-first разработка.** Можно прогнать весь сценарий без единого реального HTTP-запроса в amoCRM/YooKassa/LLM.

## Бизнес-модель (sketch, детали — в Wave 2 billing)

- Freemium: аудит и выгрузка — бесплатно, дашборды — базовые бесплатно, AI — по pay-as-you-go (YooKassa RU + Stripe EN).
- Workspace-level billing_account, ledger — в `public.billing_ledger`.

## Горизонт

- MVP-0: scaffold + документация (этот Wave).
- MVP-1: auth, workspace, CRM connect (mock), аудит, выгрузка, базовый дашборд.
- MVP-2: AI-анализ (mock LLM), research consent, retention-jobs.
- V1+: реальный amoCRM OAuth, реальный Stripe/YooKassa, реальный LLM-gateway, 2FA, live-support.
